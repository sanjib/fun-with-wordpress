# Gutenberg

Example Gutenberg Blocks

- [WooCommerce](https://wordpress.org/plugins/woocommerce/)
- [The Events Calendar](https://wordpress.org/plugins/the-events-calendar/)
- [Post Type Switcher](https://wordpress.org/plugins/post-type-switcher/)

Installing required node modules for Gutenberg development

`npm i --save-dev webpack webpack-cli babel-loader @babel/preset-react @babel/core @babel/preset-env`

Run build:

`npm run build`
